Products Configuration

Select which Plaid products to include in your Link token. Products marked as required will always be requested, while optional products may be skipped if not supported.

Column Explanations (Products can only be in ONE category):

  • Include: Products in the main products array - these are the primary products your app requests
  • Optional: Products in optional_products - requested if supported by the institution
  • Required if Supported: Products in required_if_supported_products - required if the institution supports them
  • Additional Consented: Products in additional_consented_products - products the user has consented to but may not be actively requested

⚠️ Each product can only be selected in one column. Selecting a different column for the same product will automatically uncheck the others.
